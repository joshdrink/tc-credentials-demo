Welcome to the demo. This prototype highlights a simplified version of what potential verified skills might look like in a Talent Cloud context. Below is a menu that allows you to jump to any step you'd like to view. To start from the beginning, click on Step 01.

Step 01

Create a Skill

User Group: Applicant

This UI showcases what it would be like for an applicant to create and submit a very basic skill credential. Please note only the "+ Add Skill" button is functional for demo purposes.

Step 02

Verify a Skill

User Group: Hiring Manager

This UI showcases how managers would indicate the assessment method used to verify the applicant's skill claim and subsequently "verify" the credential as valid. Please note only the "Design Research" skill is functional for demo purposes.

Step 03

View Verified Skills

User Group: Applicant

This UI showcases how applicants would perceive their skill once it had been validated. Please note that "View Details" for viewing metadata of a verified assessment is limited to the "Design Research" skill for demo purposes.
